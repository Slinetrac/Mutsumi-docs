# 前端

## 概述

Mutsumi 的前端采用现代 Web 技术构建，提供响应式和视觉效果出色的图片画廊体验。

## 技术栈

### Nuxt.js
- 用于构建 Vue.js 应用程序的框架
- 提供服务器端渲染功能
- 简化配置和设置流程
- 支持静态站点生成，实现最佳性能

### Vue 3
- 渐进式 JavaScript 框架
- 基于组件的架构设计
- 响应式数据绑定
- Composition API 提供灵活的组件逻辑

### Tailwind CSS
- 实用优先的 CSS 框架
- 提供常见 UI 元素的预构建组件
- 支持深色模式

## 功能特性

- 响应式设计，适配各种设备尺寸
- 流畅的动画和过渡效果
- 深色模式支持
- 无障碍访问功能
- 通过 SSR 优化性能
- 点击放大图片查看
- 图片模态框提供详细查看体验
